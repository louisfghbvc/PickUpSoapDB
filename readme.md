# 資料庫專題 - 多C一點O氣
### 成員: 賴彥丞, 鄒旻劭, 張耀仁, 陳守業, 劉育騏


# 功能
- 1.走到空氣汙染程度太高的地方會即時傳警告(自行設定聲音)給使用者 
- 2.可依個人病歷提供客製化的服務 
- 3.觀看現在地區的空汙程度，並提供提醒句子
- 4.可以鎖定特定區域觀看空汙程度
